To run the application do the following:
 * Install Node and npm.
 * 'npm install'
 * Install Bower (npm install bower -g)
 * 'bower install'
 * Install Gulp (npm install gulp -g)
 * Start the app with: 'gulp' and navigate to http://localhost:9999
